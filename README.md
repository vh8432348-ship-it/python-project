# Курс Python та Штучний Інтелект

Цей курс допоможе вам освоїти основи **Python** та застосування його у **штучному інтелекті (ШІ)**. Він включає практичні завдання та проекти для розвитку реальних навичок програмування та роботи з даними.

## Що ви вивчите

* Основи **Python**: змінні, цикли, функції, класи
* Роботу з **масивами та даними** через NumPy та Pandas
* **Комп’ютерний зір** за допомогою OpenCV
* Створення та тренування **нейронних мереж** для зображень і текстів
* Роботу з **LLM** та інтеграцію через LangChain
* Обробку та аналіз **табличних даних**

## Структура курсу

Курс складається з 6 блоків:

1. Вступ до Python та NumPy
2. Комп’ютерний зір та OpenCV
3. Нейромережі для зображень
4. Робота з LLM через LangChain
5. Нейромережі для текстів
6. Обробка та аналіз табличних даних

## Як почати

1. Клонуйте репозиторій:

```bash
git clone <URL-репозиторію>
```

2. Встановіть залежності:

```bash
pip install -r requirements.txt
```

3. Виконуйте практичні завдання та слідуйте інструкціям у блоках курсу.
